Overview

This script defines the basis of what a game object is. Properties such as gravity, collision and so on are applied here. This is a very powerful concept, as it allows you to craft game objects with a large degree of control. For example, maybe you have to have the player interact with all of the crates in the game; or that you want all coins to interact with players.

While these are simple, irrefutable concepts in any game, the functions contained within the GameObject class allow a statement to become the golden rule, instead of requiring you to write tedious code

Along side of the GameObject class are some key offspring classes. Particularly there are two classes that are necessary for most games - Player and NPC

Classes

GameObject

constructor(name:string, type:string, imageFile:string)

Creates the game object. Requires a name and type keys and file name of an image to render.

"name" and "type" are arbitrary - these are for various systems to use so they can interact with game objects -- either an individual game object or all particular game objects (such as all "crates"). "imageFile" is an image that will be located in the html folder of your final build.


function hasGravity(gravity:number = 9):GameObject

Applies a gravitation pull on an object. This is completely optional and the value you wish to apply is your choice too.

// index.ts
var crate:GameObject = new GameObject("badass", "Crate", "img/crate.png");
crate.hasGravity(9);

The above will apply a gravitation pull on the object, which will move the object downwards 9 pixels every update.


function collidesWith(name:string):GameObject

Used to create a list of game objects that you would like this game object to collide with.

// index.ts
var crate:GameObject = new GameObject("badass", "Crate", "img/crate.png");
var ball:GameOBject = new GameObject("red", "Ball", "img/ball.png");
crate.collidesWith("red");

Now the crate and any ball named "red" will collide.


function collidesWithAll(objectType:string):GameObject

Similar to the collidesWith function, but works on all types of a particular object.

// index.ts
var crate:GameObject = new GameObject("badass", "Crate", "img/crate.png");
var ball:GameOBject = new GameObject("red", "Ball", "img/ball.png");
crate.collidesWithAll("Ball");

Now the crate will collide with all types "Ball".


function update():void

Handles the updating logic of the game object. Modify this if you would like to add your own custom update logic.

Player

function hasKeyMovement(x:number, y:number, withKey:number):Player

Adds to a list of "input movements" that allow the game to know that when a key has been pressed to move the player.

// index.ts
var player:Player = new Player("badass", "img/player.png");
player.hasKeyMovement(-3, 0, Keys.left);
player.hasKeyMovement(3, 0, Keys.right);
player.hasKeyMovement(0, -3, Keys.up);
player.hasKeyMovement(0, 3, Keys.down);

Will move "player" in four different directions using the up, down, left and right keys.
