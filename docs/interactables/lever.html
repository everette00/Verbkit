Overview

A lever is for the intended interaction between the player and a object that can be "powered on" (an elevator, for example).

Levers require some connectable item to connect to, and a key to press. You can connect doors, elevators and conveyor belts to any lever.

Example

// index.ts
var lever:Lever = new Lever(300, 200, 32, 16);
lever.connectTo(bossDoor);
lever.interactsWith(Keys.enter);
lever.usedBy(player);

level("somewhere").hasLever(lever);

The above example did a few things: first, the lever was created at some point in space with 32x16 dimension; next, it was connected to some previously defined door; the key we expect the player to use this lever was set to the enter key; lastly the player was set to the specified user of this lever. We also added "lever" to some map. When the player makes his way through the game world and comes across this lever and activates it, the door to the boss will unlock and allow passage.
