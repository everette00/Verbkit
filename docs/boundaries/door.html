Doors are simplistic in design. They are wall bound. The only thing that you need to do with them is to create one and apply how the door needs to be opened.

constructor(x:number, y:number, width:number, height:number)

x: x-position of the door.

y: y-position of the door.

width: width of the door.

height: height of the door.
